# Finaps DOTNET Microservice Template

## Requirements:

`dotnet cli`

### Installation of Template

To install the template localy:

```sh
$ cd <path/to/folder>
$ dotnet new -i .
```

### Usage

```sh
$ dotnet new finapsms -n <ProjectName> [OPTIONS]
```

If `<projectName>` is left empty it will take the folder name.

Options:

```
-M  add mongo
-H  add HTTP
```
